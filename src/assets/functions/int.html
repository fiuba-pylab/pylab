Función int() en Python

La función int() en Python se utiliza para convertir un número o cadena de texto en un número entero. Puede manejar tanto números decimales como cadenas de texto y permite la conversión desde diferentes bases numéricas, como binario, octal y hexadecimal.

Sintaxis Básica

La sintaxis de la función int() es la siguiente:

int(x, base=10)

Donde:

  • x (opcional) es el número o cadena a convertir.
  • base (opcional) es la base numérica del valor en x, por defecto 10 (decimal).

Ejemplos

Ejemplo 1: Convertir un Número Decimal

En este ejemplo, convertimos el número decimal 10.5 en un entero:

resultado = int(10.5) print(resultado) # Salida: 10

Ejemplo 2: Convertir una Cadena de Texto

Convertimos una cadena de texto numérica en un entero:

resultado = int("15") print(resultado) # Salida: 15

Ejemplo 3: Usar una Base Numérica Diferente

Convertimos un número en binario especificando la base:

resultado = int("1010", 2) print(resultado) # Salida: 10

Notas Importantes

  • Si x no es un número o una cadena con formato numérico, int() generará un error.
  • Al convertir desde una cadena, asegúrate de que la cadena sea un número válido en la base especificada.
  • Si no se especifica la base, int() asume base 10 (decimal).
